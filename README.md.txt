# Business Intelligence Sales Dashboard

## Overview
This project is a Python-based dashboard created using Streamlit, Pandas, and Plotly.

It analyzes sales data and generates business insights through interactive visualizations.

## Features
- KPI Metrics
- Monthly Sales Trend
- Category Analysis
- Region Analysis
- Smart Insights
- Executive Summary
- Downloadable Reports

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Dataset
Kaggle Superstore Sales Dataset

## Run Project

Install requirements:

pip install -r requirements.txt

Run application:

streamlit run app.py
